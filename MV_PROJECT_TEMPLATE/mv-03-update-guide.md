# MV_03 Skill 更新指南

## 背景

隨著工作流程從 10 步改為 12 步，`mv-03-image-prompt-generation` skill 的角色發生了變化。

## 更新內容

### 舊說明（v1.0）
```
MV_03: 影像生成
生成中文影像提示詞，為 AI 生圖工具提供精確的場景和視覺描述。
```

### 新說明（v2.0）
```
MV_03: 場景提示詞設計
根據歌詞內容，為每個歌詞段落設計具體的場景視覺描述。
包含場景設定、角色動作、構圖、光影、特效等要素。
輸出：2-4 個詳細的場景提示詞（每個 300-400 字）
```

## 實施步驟

### 步驟 1：更新 Skill 名稱（可選）

**當前**：`mv-03-image-prompt-generation`  
**建議改為**：`mv-03-scene-prompt-generation`

**理由**：
- 新名稱更準確反映功能（「場景提示詞」而非「影像提示詞」）
- 避免與 Step 4（九宮格）中的「生成圖片」混淆
- 與新工作流程的名稱一致

### 步驟 2：更新 Skill 說明

在 skill 的說明文檔中，更新以下內容：

**新說明內容**：

```
# MV_03: 場景提示詞設計

## 目標
根據歌詞內容，為 MV 的每個歌詞段落設計具體的視覺場景。

## 角色
你是一位擅長視覺敘事與情感鋪陳的 MV 導演。

## 任務
根據提供的【歌詞】和【主題】，完成以下：

### 任務一：分析歌詞結構
1. 識別歌詞的不同段落（Verse、Chorus、Bridge 等）
2. 確定每個段落的情感基調和視覺意象
3. 規劃整體的視覺故事線

### 任務二：設計場景提示詞
為每個主要段落（通常 2-4 個場景）編寫詳細的場景描述。

**每個場景提示詞應包含**：
- **主題**：視覺焦點、角色表情/姿勢/情緒、服裝與道具細節
- **構圖**：主體比例與位置、相機視角、畫面平衡與留白
- **場景**：背景環境、光照、氛圍、特殊細節（煙霧、水滴等）
- **風格**：整體藝術風格、色彩搭配、材質質感
- **運鏡**：鏡頭細節（50mm f/1.8 等）、景深控制

## 輸入
- 【歌詞】：完整歌詞（來自 Step 1）
- 【主題】：MV 主題和曲風

## 輸出
- 2-4 個場景提示詞
- 每個提示詞：300-400 字
- 視覺故事線文檔（說明場景間的轉換邏輯）

## Tips
- 第一個場景通常是「開場」，建立整體視覺風格
- 中間場景應有視覺變化，保持觀眾興趣
- 最後場景應有視覺高潮或呼應開場
- 場景之間應有視覺承接（例：顏色、光線、角色位置）

## 檢查清單
- ✓ 場景數量合理（2-4 個，與歌詞段落相符）
- ✓ 每個提示詞字數足夠（300-400 字）
- ✓ 包含完整的視覺要素（主題、構圖、場景、風格、運鏡）
- ✓ 視覺風格一致
- ✓ 場景間有邏輯轉換說明
```
