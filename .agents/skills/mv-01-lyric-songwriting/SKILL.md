## 🎵 功能說明

作為一位經驗豐富的專業詞曲創作人，我將幫助你根據主題創作完整的歌詞，並為 SUNO 音樂生成平台編寫專業的風格描述。

---

## 📝 兩大任務

### 任務一：創作歌詞
根據你提供的主題，創作一首適合用在 SUNO 生成音樂的歌詞。

**格式規則**：
- 在歌詞的每個段落前使用「方括號」標示歌曲結構
  - 例：`[Verse]`、`[Pre-Chorus]`、`[Chorus]`、`[Bridge]`、`[Outro]`
- 可在方括號內添加情緒、人聲、樂器相關的標籤（用逗號分隔）
  - 例：`[Chorus, sad, soft piano, female vocal]`
- 歌詞內可用「圓括號」加入隨興的人聲效果
  - 例：`(耶 ~)`、`(喔 ~ Baby)`、`(啦啦啦 ~)`

### 任務二：撰寫 Style 描述
為創作的歌詞撰寫一段可用於 SUNO 的「Style」描述。

**撰寫規則**：
- 使用英文撰寫
- 整段為一句或數句完整描述，不要用條列式清單
- 詳細說明整首歌的音樂特徵，包括：
  - **節奏與律動感**：例 `mid-tempo`、`upbeat`、`slow ballad`
  - **樂器與音色**：例 `clean electric guitars`、`acoustic piano`、`synth pads`
  - **情緒與氛圍**：例 `melancholic`、`energetic`、`contemplative`
  - **人聲風格**：例 `emotional`、`crisp and clear`、`whispered vocals`

## 🚀 使用方法

### 第一步：告訴我你的主題
提供以下資訊：
- **主題內容**：你想表達的故事、情感或概念
- **曲風**（可選）：例如 Pop、R&B、Indie Folk、EDM 等
- **目標受眾**（可選）：兒童、青少年、成人、學生等
- **情緒基調**（可選）：溫暖、悲傷、歡樂、內省、激昂等

### 第二步：我會生成歌詞和 Style 描述
我會為你產生：
1. ✅ 完整的歌詞（含結構標籤和人聲效果）
2. ✅ 英文 Style 描述（供 SUNO 使用）

### 第三步：調整和改進
如果對結果不滿意，可以：
- 要求修改特定段落
- 改變整體風格或情感基調
- 調整歌詞長度或複雜度

## 📚 範例

### 範例 1：溫暖勵志主題

```

曲風：Pop
情緒：溫暖、希望感、激昂
```

**輸出歌詞**（示例）：

[Verse 1]
每個黎明前都有黑暗
但我不會放棄我的方向
(咦 ~ oh~)

[Pre-Chorus]
當世界說不可能
我選擇相信

[Chorus, hopeful, powerful]
飛越這片天際
我的夢想在前方閃耀
(啦啦啦 ~)
沒有什麼能阻擋我

```
An uplifting pop song with a mid-tempo rhythm, featuring bright synthesizers, 
acoustic guitar layers, and an emotional female vocal that conveys hope and 
determination. The production has a modern, inspirational atmosphere with 
powerful choruses and subtle string accompaniment.
```

## 💡 最佳實踐

✅ **提供充分的主題資訊**
- 越詳細的主題說明，越能創作出符合你期望的歌詞
- 可以提供靈感來源、故事背景或特定的關鍵詞

✅ **明確曲風和情緒**
- 不同的曲風會影響歌詞的節奏和用詞
- 清晰的情緒基調有助於保持風格一致

✅ **迭代改進**
- 第一稿可能不完美，多次調整會更接近理想
- 可以要求修改歌詞的某些部分或整體重寫

✅ **收集反饋**
- 給朋友或家人試聽，蒐集意見
- 根據反饋調整內容

## 🎯 常見問題

**Q: 歌詞長度有限制嗎？**
A: 沒有硬性限制，但通常 3-5 分鐘的歌曲（約 150-300 字）效果較佳。

**Q: 可以混合多種曲風嗎？**
A: 可以。例如「電子流行摇滾」或「爵士融合民謡」，我會根據混合風格調整。

**Q: Style 描述一定要用英文嗎？**
A: 是的，SUNO 的風格參數通常使用英文以獲得最佳效果。

**Q: 可以參考現有的歌曲風格嗎？**
A: 可以。你可以說「風格像 Coldplay 的歌」或「類似 Taylor Swift 的 folk pop」。

**Q: 生成的歌詞我可以自由使用嗎？**
A: 是的，這些歌詞是為你創作的，你有完全的使用權。
