## 🚀 快速啟動

### 場景 A：完整 MV 製作（全流程）

**時間投入**: 約 4-8 小時（含等待生音樂、生圖、生影）

**步驟**：
1. 用 **MV_01** 創作歌詞和 SUNO 風格描述
2. 用 **ai-media-generator** 在 SUNO 生成音樂
3. 用 **MV_02** 設計主角造型
4. 用 **MV_03** 為場景寫影像提示詞
5. 用 **MV_04** 生成九宮格分鏡圖
6. 用 **MV_06** 轉換成影片提示詞
7. 用 **ai-media-generator** 批量生成圖片和影片
8. 用 **YouTube 發布工具** 自動上傳到 YouTube

### 場景 B：高效製作（推薦流程）

**時間投入**: 約 5-7 小時

**步驟**：
1. ✅ **MV_01** → 創作歌詞和風格
2. 🎵 **ai-media-generator** → SUNO 生成音樂（**並行等待**）
3. ✅ **MV_02~06** → 完成所有視覺設計
4. 🎨 **ai-media-generator** → 批量生成所有素材
5. 📤 **YouTube 發布工具** → 自動上傳並發布

### 場景 C：簡報或品牌素材製作

**時間投入**: 約 1-3 小時

**步驟**：
1. 用 **MV_02** 或 **MV_07** 製作 Q 版角色
2. 用 **MV_03** 撰寫視覺描述
3. 用 **ai-media-generator** 生成高品質圖片
4. 用 **MV_05** 截圖和放大重點
5. 完成設計並匯出

### 場景 D：特殊視角製作（高級）

**時間投入**: 約 2-4 小時

**步驟**：
1. 用 **MV_08** 生成空拍視角和運鏡指令
2. 用 **MV_09** 設計首尾幀無縫轉場
3. 用 **MV_06** 為轉場加入動態效果
4. 用 **ai-media-generator** 生成最終高品質影片

## 📚 各 Skill 模組詳細說明

### MV_01：歌詞與曲風構思
- **觸發詞**: `歌詞`、`作詞`、`曲風`、`SUNO`
- **主要功能**: 根據主題創作歌詞，為 SUNO 編寫風格描述
- **輸入**: 主題、曲風、情緒
- **輸出**: 完整歌詞 + 英文風格描述

### MV_02：主角造型設計
- **觸發詞**: `角色`、`造型`、`人物設計`、`主角`
- **主要功能**: 設計 MV 主角的視覺形象（英文+中文提示詞）
- **輸入**: 性別、曲風、歌詞內容
- **輸出**: Split Screen 的人物設計圖（臉部特寫+全身照）

### MV_03：影像提示詞生成
- **觸發詞**: `生圖`、`影像提示詞`、`畫面描述`
- **主要功能**: 生成詳細的中文影像提示詞
- **輸入**: 場景概念、主題、或圖片
- **輸出**: 中文影像提示詞（含構圖、場景、風格、鏡頭等細節）

### MV_04：九宮格分鏡設計
- **觸發詞**: `分鏡`、`九宮格`、`故事板`、`MV 分鏡`
- **主要功能**: 生成 9 張多角度分鏡圖
- **輸入**: 人物設計圖 + 主題描述
- **輸出**: 9 張視覺一致的分鏡圖

### MV_05：截圖放大提質
- **觸發詞**: `裁圖`、`放大`、`提質`、`截圖`
- **主要功能**: 從九宮格中截取特定格子並放大
- **輸入**: 九宮格圖片 + Row/Column 位置
- **輸出**: 放大且提質的單張圖片

### MV_06：影片提示詞生成
- **觸發詞**: `影片提示詞`、`動態效果`、`運鏡`
- **主要功能**: 為靜態圖加入動態效果和運鏡描述
- **輸入**: 靜態圖片或主題
- **輸出**: 中文影片提示詞（含角色動作、環境變化、運鏡方式）

### MV_07：Q 版圖片設計大師
- **觸發詞**: `生圖`、`Q 版`、`插圖`、`簡報圖片`
- **主要功能**: 製作可愛的 Q 版插圖
- **輸入**: 簡報主題、角色、情緒
- **輸出**: Q 版插圖的提示詞 + 生成結果

### MV_08：空拍圖與運鏡生成
- **觸發詞**: `空拍`、`無人機`、`運鏡`、`Orbit`
- **主要功能**: 生成無人機視角，提供 Orbit/Push In/Pull Out 等運鏡指令
- **輸入**: 原始圖片 + 運鏡方式選擇
- **輸出**: 空拍視角圖片 + 運鏡指令

### MV_09：Google Flow 微電影製作
- **觸發詞**: `Google Flow`、`轉場`、`微電影`
- **主要功能**: 利用首尾幀製作無縫轉場效果
- **輸入**: 主題 + 角色類別
- **輸出**: 可直接使用的圖片和影片提示詞

### AI 多媒體生成器
- **觸發詞**: `生圖`、`生影片`、`AI 繪圖`、`Dreamina`、`FLUX`、`Kling`
- **主要功能**: 
  - 🎵 使用 SUNO 生成原創音樂（從 MV_01 的風格描述）
  - 🎨 生成高品質圖片（FLUX.1 或 Ideogram，$0.03-0.05/張）
  - 🎬 生成動態影片（Kling/Hailuo 快速版 或 Seedance 2.0 電影感，$0.07-0.30/秒）
- **使用時機**：
  1. 步驟 2：生成音樂（基於 MV_01 的 SUNO 風格）
  2. 步驟 8：生成所有圖片和影片素材

### MV_10：電影感打光指令集
- **觸發詞**: `打光`、`lighting`、`電影感`、`照明`
- **主要功能**: 提供 20 種電影感打光風格參考表
- **輸入**: 打光風格名稱或編號
- **輸出**: 英文指令 + 中文說明 + 視覺效果說明

### YouTube 自動發布工具
- **觸發詞**: `上傳 YouTube`、`YouTube 發布`、`上傳影片`、`批次上傳`
- **主要功能**：
  - 📤 單支上傳影片到 YouTube
  - 📋 批次上傳多支影片（用 CSV 控制）
  - 🎬 自動設定標題、描述、標籤、分類、隱私等
  - 📊 管理播放清單、查看頻道分析
- **使用時機**：步驟 13，發布最終 MV 到 YouTube
- **成本**：免費（YouTube Data API，有日配額限制 10,000 單位）

## ❓ 常見問題

### Q1: 我可以跳過某些步驟嗎？
**A**: 可以。根據你的目標，可以選擇需要的模組。例如只做簡報圖可跳過分鏡步驟。

### Q2: 每個步驟需要多長時間？
**A**: 取決於生圖/生影的速度。通常 MV 完整流程需要 3-5 小時（包含等待時間）。

### Q3: 用什麼工具生圖和生影？
**A**: 使用 `ai-media-generator` Skill（支援 FLUX.1、Ideogram 等）和影片工具（Kling、Hailuo、Seedance）。

### Q4: 如果對中間結果不滿意怎麼辦？
**A**: 可以重新執行該步驟的 Skill，調整提示詞或參數，直到滿意為止。

### Q5: 完成後如何上傳 YouTube？
**A**: 使用 `yt-auto-publisher` Skill 進行自動上傳和管理。

## 💡 最佳實踐

✅ **開始前**
- 準備好完整的主題和靈感
- 清晰定義 MV 的風格和受眾
- 列出關鍵視覺元素

✅ **執行中**
- 保存每個步驟的產出結果（圖片、提示詞等）
- 定期檢查視覺一致性
- 及時調整不滿意的部分

✅ **完成後**
- 檢查最終影片的品質和節奏
- 撰寫吸引人的 YouTube 標題和描述
- 分享到相應的平台並蒐集反饋
