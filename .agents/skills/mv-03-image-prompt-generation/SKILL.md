## 🎬 功能說明

作為擅長視覺敘事與情感鋪陳的 MV 導演，我將根據你提供的概念、歌詞或圖片，生成一段詳細的**中文影像提示詞**。這些提示詞可以直接用於 AI 生圖工具，幫助你精確控制生成的視覺效果。

---

## 🎨 五大設計維度

### 1. 主題（Subject）
詳細描述視覺焦點和角色元素：
- **主體**：誰或什麼是畫面的焦點？
- **表情與姿勢**：角色的面部表情、肢體語言、動作
- **情緒**：傳達什麼樣的情感狀態？
- **服裝與道具**：具體的衣著、配飾、手持物品

### 2. 構圖（Composition）
控制視覺框架和平衡：
- **主體位置**：畫面中的位置（左側、中央、右側）
- **比例與大小**：主體在畫面中佔據的大小
- **相機視角**：特寫、中景、遠景、仰視、俯視
- **平衡與留白**：畫面的視覺平衡和空間感

### 3. 場景（Environment）
描述背景和環境氛圍：
- **背景環境**：室內、戶外、自然、城市、抽象等
- **光照**：自然光、霓虹燈、月光、日光、人工光
- **氛圍**：明亮、暗沉、神祕、溫暖、冷酷等
- **特殊細節**：煙霧、水滴、粒子、雨水、光影等

### 4. 風格（Style）
統一整體的藝術方向：
- **藝術風格**：寫實、插畫、油畫、3D、動漫、電影等
- **色彩搭配**：暖色調、冷色調、高飽和、低飽和、單色等
- **材質質感**：光滑、粗糙、柔軟、堅硬、透明、反光等
- **整體氛圍**：優雅、粗獷、夢幻、荒涼、繁華等

### 5. 鏡頭（Camera）
控制專業攝影效果：
- **焦距**：50mm、85mm、200mm 等（影響景深和視角）
- **光圈**：f/1.8（淺景深）到 f/22（深景深）
- **景深效果**：背景模糊度、焦點清晰度
- **其他效果**：運動模糊、光暈、色差等

## 🚀 使用方法

### 第一步：提供輸入信息

選擇以下方式之一：

**方式 A：文字概念**
```
- 場景描述：一個人在雨夜的城市街道上
- 人物：穿著紅色雨衣的女性，表情迷茫
- 情感基調：孤獨、失落、寂寞
```

**方式 B：歌詞段落**
- 提供 MV_01 的歌詞片段
- 或者整首歌詞的特定部分

**方式 C：上傳圖片**
- 提供參考圖片（來自 MV_02 的角色設計或其他靈感圖）

### 第二步：我會生成詳細提示詞

輸出內容包含：
1. ✅ **中文影像提示詞**
   - 完整、詳細、可直接使用
   - 涵蓋主題、構圖、場景、風格、鏡頭五大維度

2. ✅ **視覺構思說明**
   - 為何這樣設計
   - 重點視覺元素解析
   - 與歌詞的聯繫

### 第三步：調整和優化

如果對提示詞不滿意，可以：
- 調整特定的視覺元素（例：改變光照或色調）
- 加強或減弱某些情感表現
- 改變構圖或視角
- 添加或移除特定的細節

## 📚 範例

### 範例 1：內省的夜間場景

**輸入**：
```

歌詞：「在黑暗中我尋找答案」
色調：冷色調、寧靜、沉思

**輸出提示詞**：
```

她的表情沉靜而思考，眼神向遠方，雙手輕輕放在膝蓋上。背景是現代化的房間，

採用特寫鏡頭，焦點集中在人物臉部，背景柔和虛化。氛圍寧靜、內省、充滿悲欣交集的情感。
電影級質感，光影細節豐富，彷彿電影場景截圖。
```

構圖採用低角度仰拍，增強了人物的氣勢和自由感。背景因奔跑產生運動模糊，
強調速度感。整體風格：電影感十足，色彩飽和度高，青春躍動的氛圍。
```

## 💡 最佳實踐

✅ **與歌詞內容保持一致**
- 提示詞應該視覺化歌詞的意象
- 情感和氛圍要相匹配

✅ **使用具體且生動的描述**
- 避免模糊或抽象的詞彙
- 用顏色、材質、光線等具體視覺元素

✅ **考慮 AI 工具的能力**
- 不同工具對複雜概念的理解不同
- 簡潔而清晰的描述效果最佳

✅ **分層描述**
- 先描述主體和主要動作
- 再加上背景和環境
- 最後添加色調和風格

✅ **參考電影術語**
- 使用專業術語如「景深」「構圖」「色溫」等
- 幫助 AI 更精確地理解你的意圖

## 🎯 常見問題

**Q: 提示詞越長越好嗎？**
A: 不一定。清晰簡潔的提示詞往往比冗長的效果更好。關鍵是描述得夠具體。

**Q: 可以混合多個參考風格嗎？**
A: 可以。例如「電影質感的插畫風」或「寫實與超現實的融合」。

**Q: 提示詞是否適用於所有 AI 生圖工具？**
A: 大部分相似，但不同工具可能需要微調。關鍵詞和風格描述通常是通用的。

**Q: 如何避免生成重複或相似的圖片？**
A: 在提示詞中加入變體細節，如角度、時間（早晨/夜晚）、季節、天氣等。

**Q: 可以要求特定的藝術家風格參考嗎？**
A: 可以，但要注意版權。建議用「風格類似」而不是「模仿特定藝術家」。
