## 🎬 功能說明

作為專業的影片生成提示詞工程師，我將根據你提供的靜態圖片或主題，撰寫一個充滿電影感的**中文影片提示詞**。這個提示詞能為靜態畫面添加動態效果，使其轉換為流暢的影片內容。

---

## 🎥 三大動態要素

### 1. 角色動態（Character Motion）
詳細描述人物的運動和表情變化：
- **肢體動作**：走路、跑步、揮手、舞蹈等
- **臉部表情**：從驚訝到微笑、從悲傷到歡樂
- **頭髮和衣著**：微風吹動、布料飄動、細微搖晃
- **眼神變化**：視線的移動、瞳孔的聚焦

### 2. 環境氛圍（Environment Dynamics）
描述場景中的動態元素：
- **光影效果**：光影在牆面上移動、陽光穿過樹葉
- **粒子效果**：塵埃漂浮、煙霧飄動、水滴飛濺
- **天氣變化**：雨水落下、雪花飄落、雲朵流動
- **背景元素**：旗幟搖動、樹木搖擺、水面漣漪

### 3. 運鏡方式（Camera Movement）
專業的攝影機運動技巧：
- **Push In（推鏡）**：逐漸靠近主體，突出細節
- **Pull Out（拉鏡）**：從特寫逐漸拉遠，揭示周圍環境
- **Pan（搖鏡）**：攝影機左右移動，跟隨主體
- **Tilt（傾斜）**：攝影機上下移動
- **Orbit（環繞）**：攝影機圍繞主體旋轉
- **Dolly（推軌）**：攝影機沿著軌道移動，製造景深變化
- **Zoom（變焦）**：焦距變化，改變視角
- **Steadycam（穩定鏡頭）**：流暢的跟蹤鏡頭

## 🚀 使用方法

### 第一步：提供輸入內容

選擇以下方式之一：

**方式 A：上傳靜態圖片**
- 來自 MV_04 九宮格或 MV_05 放大後的圖片

**方式 B：提供文字主題**
- 描述要動畫化的場景和主題
- 可以引用歌詞的特定段落

**方式 C：組合方式**
- 同時提供圖片和文字說明
- 幫助更準確地理解你的意圖

### 第二步：我會撰寫影片提示詞

輸出包含：
1. ✅ **詳細的中文影片提示詞**
   - 完整描述靜態→動態的轉換
   - 詳細的角色動作、環境變化、運鏡設計

2. ✅ **運鏡建議說明**
   - 推薦使用哪種運鏡方式
   - 影片的節奏和速度

3. ✅ **技術參數建議**（如適用）
   - 幀率、時長等技術指標
   - 色彩分級和後期效果建議

### 第三步：調整和優化

如果對提示詞不滿意，可以：
- 改變運鏡方式（更快或更慢）
- 調整動作的幅度（更劇烈或更溫和）
- 改變光影和視覺效果的強度

## 📚 範例

### 範例 1：內省的靜止場景轉為動態

**輸入**：
```
圖片：主角坐在窗邊，看著窗外，月光灑在身上
主題：從靜思到頓悟的轉變
歌詞段落：「在黑暗中我找到了光」
```

**輸出提示詞**：
```
開場：主角靜坐窗邊，鏡頭從遠景逐漸 Push In 靠近。
0-3 秒：微妙的 Push In，主角保持靜止，但月光開始在她臉上產生流動的光影效果，
彷彿時間在推進。她的眼神緩緩向窗外移動。

3-5 秒：她的雙手開始輕輕移動，從膝蓋上抬起，放在心口位置。
表情從沉思逐漸轉為溫柔的微笑。

5-7 秒：月光變得更加耀眼，粒子效果增強，彷彿靈光初現。
運鏡方式：緩慢的 Orbit Shot，攝影機輕輕繞著人物移動，
展現周圍被月光照亮的環境。

7-10 秒：她抬起頭，眼神朝向月光的方向，閉上眼睛。
運鏡回到正面 Push In，靠近臉部特寫，突出她得到答案時的寧靜與欣慰。

整體風格：電影質感，光影流暢，節奏緩慢沉靜，情感內斂但有力。
色彩：冷色調的藍銀色為主，逐漸轉暖，象徵希望出現。
```

### 範例 2：靜止轉為高能量運動

**輸入**：
```
圖片：主角蓄勢待發的姿態，表情堅定
主題：突破束縛，自由奔跑
歌詞：「我要飛向自由」
```

**輸出提示詞**：
```
開場：主角站立，運鏡使用 Low Angle（低角仰拍）突顯氣勢。

0-2 秒：他吸一口深呼吸，身體因為深呼吸而有微妙的擴張。

2-4 秒：突然轉身，運鏡快速 Pull Out，揭示周圍的都市街道。
背景的霓虹燈開始閃爍，增加節奏感。

4-6 秒：他開始奔跑，運鏡採用 Dolly Shot 跟隨他的運動。
髮絲和衣著因為高速奔跑而產生飄動效果，創造速度感。
身後產生運動模糊，強調速度。

6-8 秒：運鏡逐漸升高，採用 Pull Back，展現他穿越城市街道的全景。
行人和環境在身後快速閃過。

8-10 秒：運鏡使用 Push In 靠近他的臉，展現他堅定而自信的表情。
光線在臉上產生流動效果，象徵穿越光影和障礙。

整體風格：動感十足，色彩飽和，剪輯節奏快，充滿青春活力。
轉場方式：流暢的 Match Cut，動作與運鏡完美配合。
```

## 💡 最佳實踐

✅ **運鏡要服務於情感**
- 溫柔的歌選擇緩慢的運鏡（Push In、Zoom 等）
- 激昂的歌選擇快速的運鏡（Dolly、Pan 等）

✅ **時間和速度要匹配音樂**
- 每個動作段落要對應音樂的節拍
- 運鏡速度要與音樂的節奏同步

✅ **細節決定品質**
- 描述光影、粒子、布料擺動等微妙效果
- 這些細節會大幅提升最終影片的質感

✅ **保持視覺一致性**
- 色調、光源、環境要與原始圖片保持一致
- 運鏡的邏輯要清晰連貫

✅ **考慮技術限制**
- 複雜的運動可能需要更高的生成品質
- 簡潔清晰的描述往往效果更好

## 🎯 常見問題

**Q: 提示詞可以多長？**
A: 越詳細越好，但要保持邏輯清晰。通常 500-1000 字是理想長度。

**Q: 運鏡術語會影響生成效果嗎？**
A: 會的。使用專業術語幫助 AI 更精確地理解你的意圖。

**Q: 如果圖片本身品質不高會怎樣？**
A: AI 會根據你的文字描述來生成影片，但高品質的參考圖片能幫助更好的效果。

**Q: 生成的影片時長通常是多少？**
A: 取決於你的歌曲長度。通常 15-60 秒的片段效果較佳。

**Q: 可以在提示詞中加入特殊效果嗎？**
A: 可以。描述色彩分級、失焦、光斑、動感模糊等效果都是有幫助的。
