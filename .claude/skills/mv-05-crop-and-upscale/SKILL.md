## 🔍 功能說明

這個 Skill 幫助你從 **MV_04 九宮格分鏡設計** 的 9 張圖片中，選擇最重要或最亮眼的單張圖片，進行截圖、放大和畫質提升。適合製作重點海報、宣傳物料或特別強調的 MV 片段。

---

## 🎬 用途

### 主要應用場景

1. **MV 海報製作**
   - 選擇最具代表性的分鏡格子
   - 放大為完整尺寸，製作宣傳海報

2. **片段強調**
   - 突出高潮或重點片段
   - 製作吸睛的社群媒體貼文

3. **細節展示**
   - 放大展示角色的服裝和表情細節
   - 突出 MV 中的關鍵視覺元素

4. **質量改進**
   - 提升低解析度圖片的畫質
   - 準備高品質的最終輸出

## 🚀 使用方法

### 第一步：提供九宮格圖片

上傳來自 **MV_04** 的九宮格分鏡圖。

### 第二步：選擇要放大的格子

告訴我你要截取的位置，使用**行列標記法**：

```
九宮格位置標記：

[Row 1, Col 1]  [Row 1, Col 2]  [Row 1, Col 3]
[Row 2, Col 1]  [Row 2, Col 2]  [Row 2, Col 3]
[Row 3, Col 1]  [Row 3, Col 2]  [Row 3, Col 3]
```

**例如**：
- `Row 1, Column 1` → 左上角
- `Row 2, Column 2` → 中央（最重要的位置）
- `Row 3, Column 3` → 右下角

### 第三步：我會執行三個操作

1. ✅ **精確截圖**
   - 從九宮格中提取指定的單格圖片

2. ✅ **放大至全尺寸**
   - 將截取的圖片擴大至合適的尺寸
   - 保持長寬比例

3. ✅ **提升畫質**
   - 使用超解析度技術提升圖片清晰度
   - 減少失真和噪點

## 📚 常見放大對象

### 1. 情感特寫（最常見）
適合選擇：
- 主角面部的特寫（Row 2/3, Column 2 通常是特寫位置）
- 展現情感變化的關鍵時刻
- 淚水、微笑、驚訝等強烈表情

**應用**：
- 製作宣傳海報
- 社群媒體縮圖
- MV 預告截圖

### 2. 視覺亮點
適合選擇：
- 色彩最鮮豔的格子
- 構圖最完美的角度
- 光影最戲劇性的時刻

**應用**：
- 視覺設計參考
- 宣傳物料主圖
- 頻道封面

### 3. 細節展示
適合選擇：
- 微距鏡頭（通常在右下方區域）
- 展現服裝和配飾細節的格子
- 道具或環境的特寫

**應用**：
- 展示角色造型
- 突出設計細節
- 幕後花花公子

## 💡 最佳實踐

✅ **選擇故事中的關鍵時刻**
- 不要只選擇最漂亮的，也要選擇最有意義的
- 想想哪一格最能代表整首歌的核心

✅ **考慮使用目的**
- 海報用：選擇視覺衝擊力強的
- 社群貼文：選擇能吸引注意力的
- 預告片：選擇最神祕或最懸念的

✅ **確保圖片品質**
- 提升前的原圖品質越好，提升效果越好
- 避免選擇過度壓縮或模糊的格子

✅ **多次試驗**
- 可以選擇多個不同的格子進行放大
- 找出最適合你需求的版本

## 🎯 常見問題

**Q: 放大後會失去清晰度嗎？**
A: 現代的超解析度技術能很好地保持清晰度。原始品質越好，放大效果越好。

**Q: 放大的尺寸有上限嗎？**
A: 理論上沒有硬性上限，但建議在 4K（3840×2160）以下以獲得最佳效果。

**Q: 提升後的圖片可以用於商業用途嗎？**
A: 取決於原始圖片的版權。建議確認你有使用權。

**Q: 可以同時放大多個格子嗎？**
A: 可以。一次操作放大一個，需要多個時可以多次請求。

**Q: 如果九宮格本身品質不好怎麼辦？**
A: 超解析度技術有限度。建議先改進原始九宮格的品質，再進行放大。
